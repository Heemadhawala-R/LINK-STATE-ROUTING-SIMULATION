# LINK-STATE-ROUTING-SIMULATION
DSA(mini project)
Simulated link-state routing using Dijkstra’s algorithm
Implemented dynamic network topology creation and routing table updates
Key functionalities:
1.Network topology creation
2.Routing table initialization
3.Link cost updates and route recalculation
Routing table display
Concepts Used: Adjacency Matrix, Distance Table, Routing Table, Graph ADT, Dijkstra’s Algorithm
